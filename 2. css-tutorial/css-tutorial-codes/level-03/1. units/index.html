Units in CSS

  • CSS has several different units for expressing length.
  • Many CSS properties use "length" values, such as width, margin, padding, font-size, etc.

  • There are two types of length units:
    • Absolute Units (px) - Fixed-size units independent of other factors.

    • Relative Units (%, em, rem) - Units based on other properties that can adapt better to different screen sizes.
      • Relative length units scale relative to another length property.
      • Unit Description
        % Relative to the parent element.
        em Relative to the font-size of the element (2em means 2 times the size of the current font).
        rem Relative to the font-size of the root element.
